# hotswap-platform-ui

React + TypeScript -konsolinäkymä [HotSwap Platform](https://github.com/HotSwapPlatform) -sovelluksille.

## Käynnistys

```bash
npm install
npm run dev
```

Sovellus käynnistyy osoitteessa `http://localhost:5173`.

## Muut komennot

```bash
npm test          # testit watch-tilassa
npm run test:run  # testit kertaalleen (CI)
npm run build     # tuotantokäännös
npm run lint      # ESLint
```
